# Bluebird Web

React front-end for Bluebird Cabinetry and Design

## Commands

### `yarn start`

Start the development server.

### `yarn build`

Build the production files.

### `yarn test`

Start the test runner.

### `yarn plop <generator>`

`yarn plop <generator>`

#### `yarn plop`

Pick a generator and run it.

#### `yarn plop component`

Generate a component.
